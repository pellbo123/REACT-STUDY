# react-study

#Introduction
React 학습 과정을 날짜별로 정리하고 코드를 공유합니다

#Getting Started
1. Repository Clone
```bash
   git clone~~~
```
2. install npm packages
```bash
npm install
```

3. Start Development Server
```bash
npm start
```

4. Open Browser
```
http://localhost:3000
```
